#CALL调用新格式说明

#CALL调用支持传统方式和新方式两种格式，该功能可以在M2上的 参数设置 - 环境设置 - #CALL调用设置 上进行选择

两种方式的调用方法都是一样的： #CALL [\\XXX\XXX\XXX.txt] @XXX
区别在于M2翻译后的内容

传统方式：

  文件1.txt内容：
    [@main]
    #CALL [\\XXX\2.txt] @test

  文件2.txt内容:
    [@test]
    {
    #IF
    #ACT
    SENDMSG 1 test
    }

  M2翻译后的内容
    [@main]
    #ACT
    GOTO @test

    [@test]
    #IF
    #ACT
    SENDMSG 1 test


新方式：

  文件1.txt内容：
    [@main]
    #CALL [\\XXX\2.txt] @test

  文件2.txt内容:
    [@test]
    {
    #IF
    #ACT
    SENDMSG 1 test
    }

  M2翻译后的内容
    [@main]
    #IF
    #ACT
    SENDMSG 1 test

　

另外除了脚本文件外，部分文件也支持新方式#CALL调用设置
DefiniensConst.txt 脚本常量
MakeItem.txt 打造物品
MapInfo.txt 地图信息
MapEvent.txt 地图事件
MonGen.txt 刷怪配置
